## ENARAMP AVS Contracts

## Usage

### Build

```shell
$ forge build
```

### Deploy Eigen

```shell
$ forge script script/DeployEigenLayerCore.s.sol --rpc-url <RPC_URL> --broadcast
```

### Deploy Jackramp

```shell
$ forge script script/JackRampDeployer.s.sol --rpc-url  <RPC_URL> --broadcast
```
